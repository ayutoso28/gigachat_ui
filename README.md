# GigaChat UI

Оболочка веб-приложения для диалога с публичным API GigaChat — современные
паттерны интерфейса ChatGPT на React + TypeScript.

## Стек

- **React 19** + **TypeScript** (Vite)
- **CSS Modules** для изоляции стилей, **CSS-переменные** для тем
- **react-markdown** + **remark-gfm** для рендеринга сообщений

## Запуск

```bash
npm install
npm run dev      # dev-сервер на http://localhost:5173
npm run build    # сборка для production в ./dist
npm run preview  # локальный preview собранной версии
npm run lint     # ESLint
```
